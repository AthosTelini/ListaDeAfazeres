# Lista De Afazeres

### Esse projeto foi para um trabalho da disciplina de estrutura de dados
é uma aplicação que permite aos usuários criar listas de tarefas a serem realizadas em determinado
período de tempo. A ideia principal é ajudar o usuário a se organizar e
gerenciar suas atividades de forma mais eficiente. Ações que você deve implementar:

1. Inserir uma nova tarefa ordenada pela sua prioridade
2. Mostrar sem remover a tarefa do início da lista
3. Remover a tarefa do início e apresentar na mensagem
constando quantos dias, horas e minutos foram necessários para conclui-la.

- HTML
- CSS
- JS
